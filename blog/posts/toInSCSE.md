# 2.学生成绩数组排序(40分)
 结构体：
 ```C
 struct Student{
    char id[10];
    int score;
 }
 ```
函数原型：
```C
 void SortStudent(struct Student stu[], int n);
 ```
其中stu是学生成绩数组，n是有效数据个数，要求对给定的数组按照
 - score更大的学生排列在前；
 - 若score相等，则按照id从小到大排列

 例：

输入
 > 3
 >
 > S003 74
 > 
 > S002 85
 >
 > S001 74

 则输出（不要求printf，matu直接检测数组本身）：
 >S00285
 >
 >S001 74
 >
 >S003 74
 # 3.梅森素数(20分)
 梅森素数是指，对于一个正整数$n$，若$n$是素数，且$2^n-1$也是素数，则后者称为梅森素数，前者称为指数。例如：
$2$是素数， $2^2-1=3$是素数，则$3$是梅森素数。

输入max_n，输出**指数**不超过max_n的梅森素数，用空格隔开，最后一个数的空格后加换行，例：

输入
>5

输出
> 3 7 31 \n
